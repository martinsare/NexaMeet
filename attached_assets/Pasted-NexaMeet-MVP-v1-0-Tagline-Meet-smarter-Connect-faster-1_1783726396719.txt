Tech Stack

Frontend: TanStack Start + React 19

Language: TypeScript

Styling: Tailwind CSS

Components: shadcn/ui

State Management: TanStack Query + TanStack Store

Routing: TanStack Router

Forms: TanStack Form

Authentication: Supabase Auth

Database: Supabase PostgreSQL

Storage: Supabase Storage

Realtime: Supabase Realtime

Video & Audio: WebRTC

Media Infrastructure: LiveKit (recommended) or another SFU

AI: OpenAI APIs

Deployment: Vercel (frontend) + LiveKit deployment + Supabase

What makes NexaMeet different?

Instead of saying “We’re another meeting app,” your product promise is:

Join meetings in seconds with minimal friction.

Stay connected on poor internet through adaptive quality.

Never take manual meeting notes thanks to AI summaries and action items.

Find anything from any meeting using transcripts and AI search.

Keep the interface clean and intuitive, avoiding feature overload and make it unique and match the content of every pages
